# Esoteros Analytics  
A Next.js 14 Platform for AI-Driven Life Insights, Recovery Support, and Personal Transformation

Esoteros Analytics is a modern SaaS platform designed to help individuals‚Äîespecially those rebuilding after rehab or major life transitions‚Äîdiscover meaningful patterns in their behavior, habits, finances, and daily life.  
By combining **data analytics**, **AI-generated insights**, and **intuitive dashboards**, Esoteros helps users build structure, gain clarity, and design a healthier future.

---

## üåü Project Vision

Esoteros Analytics aims to become a **Life Operating System (Life-OS)** that empowers users to:

- Track habits, mood, energy, and behavior  
- Understand hidden correlations across their data  
- Build healthy routines and recovery momentum  
- Receive personalized AI insights and recommendations  
- Organize their life through dashboards, timelines, and structured planning  

The platform blends software development best practices with compassion-driven functionality to support individuals striving for stability, identity, and long-term growth.

## üß± Technology Stack

- **Next.js 14 (App Router)**
- **TypeScript**
- **TailwindCSS**
- **Prisma ORM (PostgreSQL)**
- **NextAuth (Auth scaffolding)**
- **Server Components + modern React patterns**
- **AI integrations** (OpenAI and future providers)

Designed for scalability, multi-tenancy, and rapid MVP evolution.

### Directory Summary

- **app/**  
  App Router pages, layouts, and application logic  
- **app/api/**  
  REST-style route handlers for auth, user data, and analytics  
- **components/**  
  Reusable UI components (Navbar, Sidebar, Cards, Modals, etc.)  
- **lib/**  
  Utility helpers (db, auth, analytics helpers)  
- **styles/**  
  Global styles + Tailwind config  
- **types/**  
  Shared TypeScript types and interfaces  
- **prisma/**  
  Schema and future seed scripts  
- **public/**  
  Static assets  
- **scripts/**  
  Maintenance and automation scripts
